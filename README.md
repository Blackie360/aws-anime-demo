# AWS Amplify Gen 2 Anime Demo

This project is a full-stack web app using **AWS Amplify Gen 2** with **Next.js 14**. It serves as an example for building modern web applications with Amplify's new features like authentication, API integration, and sandbox environments.

## Key Features

- **Next.js 14**: For a highly performant React framework.
- **AWS Amplify**: Offers backend services such as authentication, GraphQL APIs, and more.
- **Amplify Sandbox**: A testing environment for rapid backend changes.

## Setup Instructions

1. **Clone the Repository**

   ```bash
   git clone https://github.com/Blackie360/aws-anime-demo.git
   cd aws-anime-demo
   ```

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Amplify Sandbox**

   To initialize the Amplify sandbox:

   ```bash
   npx amplify sandbox
   ```

4. **Run Development Server**

   To start the development environment:

   ```bash
   npm run dev
   ```
